# Changelog

## v0.1

Initial public release.

- Added native 2-4 player local split-screen support.
- Added gamepad-focused split-screen input handling.
- Added per-player player setup and controller setup.
- Added cooperative split-screen save/load support.
- Added split-screen HUD, notification, audio, rumble, FOV, and end-of-level summary adjustments.
- Added custom split-screen menu branding and RPG HUD artwork support.
- Added safer default keyboard/mouse and gamepad controls.
