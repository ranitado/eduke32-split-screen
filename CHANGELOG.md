# Changelog

## v0.3

- Fixed auto aim not working and configuration save issues.
- Improved split-screen results screen with per-player kills/secrets.
- Fixed weapon pickup notifications and pickup sounds.
- Improved controller compatibility and first-run display handling.

## v0.2

- Improved gamepad controls, added LT precision aiming, auto-run disable option, default stick feel, and disabled view centering by default.
- Improved split-screen visual effects and fixed jetpack sound, input, and menu issues.

## v0.1

Initial public release.

- Added native 2-4 player local split-screen support.
- Added gamepad-focused split-screen input handling.
- Added per-player player setup and controller setup.
- Added cooperative split-screen save/load support.
- Added split-screen HUD, notification, audio, rumble, FOV, and end-of-level summary adjustments.
- Added custom split-screen menu branding and RPG HUD artwork support.
- Added safer default keyboard/mouse and gamepad controls.
