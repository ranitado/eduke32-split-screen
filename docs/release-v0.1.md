# Release Notes - v0.1

First public Windows x64 release of EDuke32 Split-Screen.

## Download

Use `eduke32-split-screen-v0.1-windows-x64.zip` from the GitHub release.

## Installation

Extract the ZIP to a new folder and copy your legal `DUKE3D.GRP` into that folder before running `eduke32.exe`.

## Notes

- This release does not include `DUKE3D.GRP`.
- Start with a fresh config for the intended default controls.
- Report split-screen issues with player count, controller count, map, and whether `Separate KB/M pads` is enabled.
