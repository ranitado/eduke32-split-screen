# Release Notes - v0.3

Windows x64 update for EDuke32 Split-Screen.

## Download

Use `eduke32-split-screen-v0.3-windows-x64.zip` from the GitHub release.

## Changes

- Fixed auto aim not working and configuration save issues.
- Improved split-screen results screen with per-player kills/secrets.
- Fixed weapon pickup notifications and pickup sounds.
- Improved controller compatibility and first-run display handling.

## Installation

Extract the ZIP to a new folder and copy your legal `DUKE3D.GRP` into that folder before running `eduke32-split-screen.exe`.

## Notes

- This release does not include `DUKE3D.GRP`.
- Start with a fresh config for the intended default controls.
