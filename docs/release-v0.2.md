# Release Notes - v0.2

Windows x64 update for EDuke32 Split-Screen.

## Download

Use `eduke32-split-screen-v0.2-windows-x64.zip` from the GitHub release.

## Changes

- Improved gamepad controls, added LT precision aiming, auto-run disable option, default stick feel, and disabled view centering by default.
- Improved split-screen visual effects and fixed jetpack sound, input, and menu issues.

## Installation

Extract the ZIP to a new folder and copy your legal `DUKE3D.GRP` into that folder before running `eduke32-split-screen.exe`.

## Notes

- This release does not include `DUKE3D.GRP`.
- Start with a fresh config for the intended default controls.
